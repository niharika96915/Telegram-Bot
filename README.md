# 🤖 Caption Crafter Bot (Telegram + AI API)

An AI-powered Telegram bot that generates engaging social media captions based on user input like topic, tone, and platform.

---

## 🚀 Features

* Generate **creative captions instantly**
* Supports different tones (funny, professional, emotional)
* Works for multiple platforms (Instagram, LinkedIn, etc.)
* Simple chat-based interaction via Telegram
* Powered by AI API (OpenAI / Google Generative AI)

---

## 🛠️ Tech Stack

* Python
* python-telegram-bot
* OpenAI API (or Google Generative AI)
* Async programming

---

## 📌 How It Works

1. User sends a message to the bot
2. Bot receives input via Telegram API
3. Input is sent to AI API with a structured prompt
4. AI generates captions
5. Bot replies with 2–3 caption options

---

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/caption-crafter-bot.git
cd caption-crafter-bot
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Add API Keys

Create a `.env` file:

```env
TELEGRAM_BOT_TOKEN=your_telegram_token
OPENAI_API_KEY=your_openai_key
```

### 4. Run the Bot

```bash
python bot.py
```
